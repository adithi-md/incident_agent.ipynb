# Smart Incident Prevention Agent Using Autoencoder-Based Anomaly Detection

## Overview
This project implements a smart incident prevention agent that detects system anomalies using an autoencoder. The model learns normal system behavior and identifies anomalies through reconstruction error.

## Key Features
- Encoder–decoder autoencoder architecture
- Unsupervised anomaly detection
- Reconstruction error–based thresholding
- Automated incident alert simulation
- Synthetic system metrics

## Tech Stack
- Python
- TensorFlow / Keras
- NumPy, Pandas
- Scikit-learn
- Matplotlib
